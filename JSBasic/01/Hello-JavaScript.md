### 01-1 웹 프로그래밍이란?

- 클라이언트 -> 웹 브라우저를 통해 필요한 정보 요청 -> 서버
            <-    요청 처리 후 알맞은 정보 전달    <- 
- 프론트엔드 개발 :  HTML, CSS, 자바스크립트

<br/>
<br/>

### 01-2 자바스크립트로 무엇을 할 수 있나?
- 웹 사이트를 동적으로
- 웹 브라우저에서 실행되는 프로그램
- 서버를 구성, 서버용 프로그램 : node.js

<br/>
<br/>

### 01-3 자바스크립트 특징
- 모든 웹 브라우저에서 동작
- 웹 브라우저에서 실행 결과를 즉시 확인 가능
- 다양한 프로그램 : IoT, 드론 등
- 다양한 자바스크립트 OpenAPI
- 다양한 라이브러리, 프레임워크
  - 제이쿼리
  - 리액트, 앵귤러, 뷰
